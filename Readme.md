# Sys Tools

* Installation
	
    ```
    git clone https://github.com/hotnuma/systools.git
    cd systools
    ./install.sh
    ```

* Build hoedown
    
    ```
    git clone https://github.com/hoedown/hoedown.git
    cd hoedown
    make && sudo make install
    ```
